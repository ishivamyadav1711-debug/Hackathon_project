# ⚡ Smart Energy Optimization Dashboard

## 🚀 Overview
This project is a real-time energy monitoring system.

### Features:
- Live power tracking
- Efficiency score
- Energy waste detection
- Cost estimation

---

## 🛠 Tech Stack
- Frontend: React + Chart.js + Tailwind
- Backend: FastAPI (Python)

---

## ▶️ How to Run

### 1️⃣ Clone repo
git clone https://github.com/ishivamyadav1711-debug/Hackathon_project.git

cd Hackathon_project

---

### 2️⃣ Backend Setup

cd backend

python -m venv .venv

.\.venv\Scripts\activate

pip install fastapi uvicorn

python -m uvicorn main:app --reload

---

### 3️⃣ Frontend Setup

open new terminal

cd frontend

npm install

npm run dev

---

## 🌐 Open in browser

Frontend: http://localhost:5173  
Backend: http://127.0.0.1:8000/data

## 📦 Requirements

- Python 3.10+
- Node.js 18+
